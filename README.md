# Meme Generator App 😂

A fun and interactive Meme Generator built using React while learning from Scrimba.  
This project allows users to generate random memes and add custom top and bottom text to create personalized memes instantly.

---

## 🚀 Features

- Generate random meme images
- Add custom top text
- Add custom bottom text
- Dynamic UI updates using React state
- Responsive and simple design
- Fetch memes from an API

---

Built With

- React
- Vite
- JavaScript
- HTML5
- CSS3

## ⚙️ Installation and Setup

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
```

### 2. Navigate to project folder

```bash
cd YOUR_REPO_NAME
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start development server

```bash
npm run dev
```

OR if using Create React App:

```bash
npm start
```
